# Public Static Assets

You can upload your logo files directly to this folder:
1. Name the English/Chinese title logo `logo.png` - it will render in the top navigation bar's left side.
2. Name the cute puppy drawing logo `logo2.png` - it will render in the top navigation bar's right side.

If any of these images are missing or fail to load, the system automatically falls back to our beautiful hand-drawn, high-fidelity responsive layout & SVGs.
